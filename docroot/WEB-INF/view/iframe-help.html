SSO IFrame Help

The following preferences can be used with the SSO IFrame Portlet in addition to the standard IFrame preferences

Preference Usage
sso.type The type of Single signon authentication. Valid values: form | basic | basic.preemptive | url | url.base64
sso.url.Credential The name of the credential input field or parameter
sso.url.Principal The name of the user name input field or parameter
SSO Principal The actual user name
SSO Credential The actual crendential value
